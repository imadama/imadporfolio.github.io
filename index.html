Ik ben trots op mijn prestaties en ik blijf mezelf uitdagen door nieuwe uitdagingen aan te gaan en door te leren. Ik geloof in het leveren van kwaliteitswerk en het constant verbeteren van mijn vaardigheden, en ik kijk uit naar de kans om mijn ervaring en expertise te blijven delen met anderen.

Opleiding

Recent Projects

Project 1: Python

Python

Tijdens mijn werk met Python heb ik veel geleerd over het programmeren in deze taal. In het begin van het schooljaar zijn we begonnen met de basis van Python, zoals het opzetten van Github, het creëren van een overzichtelijke mappenstructuur en het schrijven van standaard Python code. Na een maand begon ik met mijn eerste project, waarbij ik een tekstgebaseerd spel moest maken genaamd "Guess Capital game". Hier heb ik een kleine game ontwikkeld. Vervolgens kreeg ik grote opdrachten, waarbij ik alles van een while-loop tot functies moest gebruiken. Zo heb ik een rekenmachine, een loodjesverdeler, een bestelsysteem voor een ijssalon en het spel Lingo gemaakt met Python.
